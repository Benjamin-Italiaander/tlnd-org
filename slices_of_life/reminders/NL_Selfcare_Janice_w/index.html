🧩 Zelfzorg-Checklist (Dagelijks) Link to heading

Gebaseerd op inzichten uit Running on Empty

🌿 Basis lichamelijke zorg Link to heading

  • Heb ik genoeg water gedronken vandaag?
  • Heb ik op tijd gegeten (niet te weinig, niet te veel)?
  • Heb ik voeding gekozen die mij goed doet?
  • Heb ik pauzes genomen wanneer mijn lichaam moe werd?
  • Heb ik vandaag minimaal 10 minuten bewogen?

🧘‍♂️ Rust & herstel Link to heading

  • Heb ik vandaag momenten van echte rust gehad (zonder scherm)?
  • Heb ik mijn ademhaling bewust even rustig gemaakt?
  • Heb ik vandaag iets gedaan dat mijn stress verlaagt?
  • Heb ik vroeg genoeg aangegeven dat ik overprikkeld raakte?
  • Heb ik mijn avond rustig afgesloten?

💛 Emotionele check-in Link to heading

  • Heb ik vandaag stilgestaan bij wat ik voel?
  • Heb ik dat gevoel toegestaan zonder oordeel?
  • Heb ik 1–2 minuten genomen om een gevoel er gewoon te laten zijn?
  • Heb ik iets gedaan dat mijn emotie ondersteunt (i.p.v. onderdrukt)?

🔄 Grenzen & balans Link to heading

  • Heb ik “nee” gezegd waar dat nodig was?
  • Heb ik iets niet gedaan waar ik eigenlijk geen energie voor had?
  • Heb ik mensen niet boven mezelf gezet ten koste van mijn eigen welzijn?
  • Heb ik hulp gevraagd als iets te zwaar voelde?

🌼 Vriendelijkheid naar mezelf Link to heading

  • Heb ik vandaag één kleine vriendelijke actie voor mezelf gedaan?
  • Heb ik mezelf niet gekleineerd of streng benaderd?
  • Heb ik mezelf behandeld zoals ik een vriend(in) zou behandelen?
