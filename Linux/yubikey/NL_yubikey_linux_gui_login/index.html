Nl Yubikey Linux Gui Login

Hier is een duidelijke en complete handleiding om op Ubuntu in te loggen met je YubiKey (PAM + FIDO2 / WebAuthn). Dit werkt voor:

  • Ubuntu desktop login (GDM)
  • sudo
  • su
  • TTY login

Ik geef zowel de snelle versie als de uitgebreide versie.

✅ Korte versie (TL;DR) Link to heading

  1. Installeer FIDO2-tools:
sudo apt install libpam-u2f pamu2fcfg
  1. Maak de config-map aan:
mkdir -p ~/.config/Yubico
  1. Registreer je YubiKey:
pamu2fcfg > ~/.config/Yubico/u2f_keys

(raak de YubiKey aan)

  1. Activeer YubiKey login voor sudo:
sudo nano /etc/pam.d/sudo

Voeg bovenaan toe:

auth       required   pam_u2f.so
  1. (Optioneel) Activeren voor desktop login:
sudo nano /etc/pam.d/gdm-password

Bovenaan toevoegen:

auth       required   pam_u2f.so

Herstart → klaar.

🧩 Uitgebreide handleiding: Ubuntu inloggen met YubiKey (PAM + U2F/FIDO2) Link to heading

1. Installeer benodigde software Link to heading

Ubuntu heeft deze pakketten in de repo:

sudo apt install libpam-u2f pamu2fcfg

2. Configuratiemap maken Link to heading

mkdir -p ~/.config/Yubico

3. YubiKey registreren voor PAM Link to heading

Dit koppelt jouw Linux-account aan de YubiKey:

pamu2fcfg > ~/.config/Yubico/u2f_keys

Je YubiKey begint te knipperen → raak hem aan.

Wil je meerdere YubiKeys toevoegen (bijv. backup)?

pamu2fcfg -n >> ~/.config/Yubico/u2f_keys

4. Sudo beveiligen met YubiKey Link to heading

Best practice: eerst sudo beveiligen. Open PAM-config:

sudo nano /etc/pam.d/sudo

Voeg helemaal bovenaan toe:

auth required pam_u2f.so

Test in een andere terminal met:

sudo ls

Je moet nu je YubiKey aanraken.

5. Desktop login (GDM) beveiligen Link to heading

Voor Ubuntu GNOME:

sudo nano /etc/pam.d/gdm-password

Bovenaan:

auth required pam_u2f.so

6. Terminal / TTY login Link to heading

sudo nano /etc/pam.d/login

Bovenaan:

auth required pam_u2f.so

7. Opties (aanbevolen) Link to heading

Je kunt de login alleen YubiKey laten vereisen, of YubiKey of wachtwoord.

🔐 “Hard” (YubiKey verplicht) Link to heading

auth required pam_u2f.so

🛡 “Soft” (YubiKey of wachtwoord) Link to heading

auth sufficient pam_u2f.so
auth include system-auth

8. Test, voordat je login breekt Link to heading

BELANGRIJK: Open een tweede terminal of TTY (Ctrl+Alt+F3) voordat je uitlogt.

Test:

sudo ls

Als het werkt → je bent klaar.

Als niet → verwijder de rule uit PAM.
